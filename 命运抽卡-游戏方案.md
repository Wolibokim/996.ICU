# 🎴 命运抽卡 (Fate Gacha)

> **"你的人生，由卡牌决定。每一次抽卡，都是命运的转折。"**

---

## 一、30秒理解这个游戏

```
抽卡 + 人生模拟 + Roguelike = 命运抽卡

你抽到什么卡，就过什么人生。
每局3-5分钟，体验一个完整的人生。
收集100+种人生结局，看看命运有多少种可能。
```

**核心爽点**：
- 🎰 **抽卡的刺激** - 每次抽卡都心跳加速
- 🎲 **随机的人生** - 每局都完全不同
- 🧠 **策略的深度** - 用有限的牌打出最好的人生
- 📚 **收集的满足** - 解锁所有人生结局
- 🏆 **比较的快感** - 我的人生比你精彩！

## 七、付费设计

### 7.1 核心原则

```
✅ 付费加速，不付费也能玩
✅ 卖的是"体验"不是"数值"
✅ 不破坏公平性（排行榜分付费/免费）
```

### 7.2 付费点

**1. 体力系统**
```
每局消耗1点体力
免费玩家：每天5点体力（5局）
广告恢复：看广告+1点
付费购买：6元=20点体力
月卡：18元=每天10点体力
```

**2. 抽卡加成**
```
免费抽卡：普通概率
金币抽卡：100金币一次（游戏内获得）
钻石抽卡：10钻石一次（付费货币）
         → 概率提升，保底机制
```

**3. 皮肤系统**
```
人生卡牌皮肤：改变卡牌外观
界面主题：赛博朋克/古风/二次元
人生名片边框：各种炫酷边框
```

**4. 特权功能**
```
VIP特权（30元/月）：
· 每日体力上限+5
· 解锁"时间回溯"功能（可悔棋1次）
· 专属VIP皮肤
· 优先参与新活动
```

## 八、技术实现

### 8.1 核心数据结构

```typescript
// 初始卡
interface OriginCard {
  id: string;
  type: 'family' | 'talent' | 'appearance';
  rarity: 'N' | 'R' | 'SR' | 'SSR';
  name: string;
  description: string;
  effects: AttributeModifier[];
  unlockRoutes?: string[]; // 解锁的特殊路线
}

// 事件卡
interface EventCard {
  id: string;
  type: 'opportunity' | 'crisis' | 'choice' | 'relationship';
  stage: 'childhood' | 'youth' | 'adult' | 'middle' | 'senior';
  title: string;
  description: string;
  requirements?: Requirement[]; // 触发条件
  choices: Choice[];
}

// 选择
interface Choice {
  id: string;
  text: string;
  effects: Effect[];
  requirements?: Requirement[];
  nextEvents?: string[]; // 可能触发的后续事件
}

// 人生状态
interface LifeState {
  age: number;
  attributes: {
    intelligence: number;  // 智力
    charm: number;        // 魅力
    health: number;       // 健康
    wealth: number;       // 财富
    connections: number;  // 人脉
    luck: number;         // 运气
    happiness: number;    // 幸福感
  };
  achievements: string[];
  relationships: Relationship[];
  abilityCards: AbilityCard[];
  lifeEvents: LifeEvent[];
}

// 结局
interface Ending {
  id: string;
  rarity: 'N' | 'R' | 'SR' | 'SSR';
  category: 'career' | 'love' | 'family' | 'special' | 'hidden';
  name: string;
  description: string;
  epitaph: string; // 墓志铭
  conditions: Condition[];
  isUnlocked: boolean;
}
```

## 九、为什么这个游戏会火？

### 9.1 对比分析

| 对比维度 | 传统抽卡游戏 | 命运抽卡 |
|---------|-------------|---------|
| 抽什么 | 角色/装备 | **人生** |
| 一局时长 | 无尽/很长 | **3-5分钟** |
| 重玩价值 | 低 | **极高**（每局不同） |
| 分享欲望 | 炫耀角色 | **分享人生故事** |
| 情感连接 | 弱 | **强**（代入感） |
| 话题性 | 一般 | **极强**（人生选择） |

### 9.2 病毒传播点

```
1. "我抽到了XXX人生，笑死" → 搞笑传播
2. "这个选择太难了..." → 讨论传播
3. "我的人生评分98，谁能超过我？" → 挑战传播
4. "如果当初选了另一个..." → 共鸣传播
5. "你的人生vs我的人生" → 对比传播
```

### 9.3 一句话总结

> **每个人都想知道：如果人生可以重来，会怎样？**
> **《命运抽卡》让你用3分钟，体验无数种可能。**
